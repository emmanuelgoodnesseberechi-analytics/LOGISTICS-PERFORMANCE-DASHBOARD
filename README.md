Project Overview

This capstone project presents a Logistics Performance Dashboard developed in Power BI to analyze key logistics and operational metrics. The dashboard provides insights into delivery performance, operational efficiency and overall logistics effectiveness through interactive visualizations and reporting.

Objectives

* Monitor logistics and delivery performance.
* Track key operational metrics and KPIs.
* Identify trends and areas for improvement.
* Support data-driven decision-making through visual reporting.

Tools Used

* Power BI
* Power Query
* DAX
* Microsoft Excel

Key Insights

* Analyzed logistics performance across multiple operational metrics.
* Evaluated delivery efficiency and operational effectiveness.
* Monitored key performance indicators through visual reporting.
* Used slicers to enable interactive data exploration.
